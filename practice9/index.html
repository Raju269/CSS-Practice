Units

CSS has several different units for expressing a length Many CSS properties take "lenght" values, such as width, margin,padding,font-size,et.
Length is a number followed by a length unit, such as 10px ,2em etc. 

    h1{
         font-size:60px
        }
    p{
        font-size:25%;
        line-height:50em;
        }
    There are two types of lenght units: aboulute and relative.
  1. Absolute lenght

    • mm
    • in
    • px
    • pt
    • pc
  2. Relative Lenghts
    • em
    • rem
