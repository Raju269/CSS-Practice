Colors

We've already seen some properties in CSS that takes color values . Here's and example: 

    p{
            color:blue;
             }
    There are Several different ways to Specify color in CSS.
  • Color Keyword
  • RGB
  • RGBA
  • HSL
  • HSLA
  • Hexadecimal
